Site pronto para GitHub + Railway

Catálogo da Roblox Creator Store.

Pesquise modelos, áudios, decals, meshes, plugins e outros assets da Creator Store. Esta versão usa backend Node.js para evitar bloqueio de CORS.

Suba este projeto no GitHub e conecte no Railway. O Railway vai rodar o servidor e liberar um link público para o site.

Como publicar no GitHub + Railway

1. Crie um repositório no GitHub.
2. Envie todos os arquivos deste projeto para o repositório.
3. No Railway, clique em New Project.
4. Escolha Deploy from GitHub repo.
5. Selecione seu repositório.
6. O Railway vai detectar o package.json e rodar npm start.
7. Depois gere o domínio público em Settings ou Networking.
